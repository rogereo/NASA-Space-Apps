About Exoplanet Explorer

Exoplanet Explorer lets users explore how machine learning models perform on real exoplanet data from NASA’s Kepler mission. We trained four models — Gradient Boosting, Random Forest, XGBoost, and LightGBM — using Kepler Objects of Interest. Along with model results, users can view interactive 3D visualizations of planets, stars, and their orbits. For instance, the Kepler-227 system shows a Neptune-like planet orbiting a G-type star.

The project enhances its dataset by using the LightKurve API to extract raw star light data, which is then attached to the KOI entries, allowing users to watch a time-lapse 3D view of these light changes over time to reveal patterns from planets or stellar activity.
